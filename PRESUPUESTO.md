# Propuesta comercial — Fuego Amigo Catering

**Empresa:** Structura · structura.com.ar  
**Responsable:** Gastón Fernández  
**Proyecto:** Sitio web institucional + E-commerce para catering

## Objetivo

Crear un sitio web profesional que genere pedidos directos por WhatsApp, presente el catálogo de productos y servicios de forma ordenada, y permita la administración completa del contenido sin necesidad de conocimientos técnicos.

## Alcance del servicio

Diseño y desarrollo de Single Page Application (SPA) con React + TypeScript.

**Secciones:** hero, servicios, galería de eventos, tienda, productos destacados, medios de pago, combos, FAQs, contacto.

**Funcionalidades principales:**
- Catálogo de productos con filtros y búsqueda
- Carrito de compras con persistencia local
- Checkout con formulario de datos del cliente
- Generación automática de mensaje de WhatsApp con pedido completo
- Panel de administración (CRUD) para productos, eventos, promociones y FAQs
- Carga de imágenes desde el panel de administración
- CTAs a WhatsApp con mensajes prearmados por tipo de consulta
- Landing page con protección por código de acceso

**Panel de administración:**
- Gestión completa de productos (crear, editar, eliminar)
- Gestión de eventos realizados con múltiples imágenes
- Gestión de promociones y descuentos
- Gestión de preguntas frecuentes
- Importación y exportación de datos JSON para respaldo

**Pasos a seguir:**
- Adaptación de todas las secciones o fragmentos que no puedan editarse desde el CRUD
- Conversión de contenido hardcodeado a datos gestionables desde el panel
- Identificación y adaptación de textos, imágenes y configuraciones estáticas

## Beneficios clave

Más pedidos reales con mensajes guiados por WhatsApp que incluyen toda la información del cliente y su pedido.

Administración completa del contenido sin necesidad de conocimientos técnicos. El cliente puede agregar productos, eventos, promociones y FAQs desde el panel.

Presencia profesional que refleja la calidad del servicio de catering.

Carrito de compras funcional que facilita la compra y genera pedidos estructurados.
